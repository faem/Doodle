# Doodle Android App

This is a simple drawing application for Android, built using Jetpack Compose.

## Features

- **Drawing Canvas**: A full-screen canvas to draw on.
- **Pen and Eraser Tools**: Switch between a pen for drawing and an eraser for corrections.
- **Color Picker**: Choose from a variety of colors for your pen.
- **Stroke Width Picker**: Adjust the thickness of your pen's stroke.
- **Clear Canvas**: Easily clear the entire drawing with a single tap.
- **Save Drawing**: Save your drawing to your device's picture gallery in a "Doodles" folder.
- **View Saved Drawing**: After saving, a confirmation message with a "View" button appears, allowing you to quickly open the saved image.

## How to Run the App

1.  **Clone the repository** or download the source code.
2.  **Open the project** in Android Studio.
3.  **Connect an Android device** or start an emulator.
4.  **Click the "Run" button**.
